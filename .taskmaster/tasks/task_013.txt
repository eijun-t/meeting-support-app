# Task ID: 13
# Title: AIによる会議決定事項・アクションアイテム自動抽出機能の実装
# Status: done
# Dependencies: 8, 10
# Priority: medium
# Description: 会議の文字起こしデータ全体から、AIを用いて主要な決定事項と「タスク内容」のみを抽出し、リスト形式で生成するシンプルな機能を開発します。
# Details:
1. 文字起こしデータ（会議全体のテキスト）をインプットとして受け取るAPIまたはモジュールを設計します。
2. 最新の日本語対応LLM（例：OpenAI GPT-4 Turbo、Gemini 1.5 Pro等）またはカスタムプロンプトを活用し、会議の主要な決定事項とアクションアイテム（タスク内容のみ）を抽出するプロンプトエンジニアリングを行います。
3. アクションアイテムは「タスク内容」のみを持つ構造化データとして出力し、リスト形式でフロントエンドに返却できるようにします。
4. 決定事項・アクションアイテムの抽出精度を高めるため、会議アジェンダや参加者情報（Task 8で実装済み）もプロンプトに組み込む設計とします。
5. プライバシー・セキュリティ要件（外部API利用時のデータ送信範囲、ログ管理等）を考慮します。

# Test Strategy:
・多様な会議文字起こしデータを用意し、主要決定事項とアクションアイテム（タスク内容のみ）が正確に抽出・リスト化されるかを検証します。
・会議アジェンダや参加者情報をプロンプトに含めた場合の精度向上をテストします。
・外部API利用時のデータ送信範囲やプライバシー要件が仕様通りかをチェックします。
