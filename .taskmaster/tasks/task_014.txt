# Task ID: 14
# Title: Supabase上の過去議事録・関連アイテム閲覧UIの実装
# Status: done
# Dependencies: 9, 13
# Priority: medium
# Description: 「議事録一覧」機能として、Supabaseに保存された過去の議事録や関連アイテムを2カラムレイアウトで閲覧できるUIを実装する。左サイドバーに日付とタイトルの一覧を表示し、右側の本体エリアで選択した議事録の要約・決定事項・アクションアイテムを表示する。フロントエンドで動作するシンプルなワード検索機能を備え、アプリ内から「議事録一覧」ページへの導線も設ける。
# Details:
1. Supabaseから過去の議事録や関連アイテム（セッションデータ）を取得するAPI呼び出しを実装する。
2. UIは2カラムレイアウトとし、左サイドバーに「日付とタイトルの一覧」をリスト形式で表示する。
3. 右側の本体エリアには、選択した議事録の「要約」「決定事項」「アクションアイテム（Task 13の出力）」を見やすく整理して表示する。
4. 左サイドバー上部にフロントエンドで動作するワード検索ボックスを設置し、タイトル・要約・決定事項・アクションアイテムのいずれかに該当する議事録のみを絞り込めるようにする。
5. アプリ内のナビゲーションやメニュー等に「議事録一覧」ページへの導線を追加する。
6. Supabaseのデータ構造や取得APIはTask 9の実装に準拠し、認証なしでアクセス可能とする。
7. データ取得中のローディング表示や、データが存在しない場合の空状態メッセージも実装する。
8. コードの再利用性を高めるため、既存UIコンポーネントのリファクタリングも検討する。

# Test Strategy:
・Supabaseに保存済みの複数の議事録・関連アイテムが左サイドバーに日付とタイトルの一覧として正しく表示されることを確認する。
・サイドバーの検索ボックスでワード検索した際、該当する議事録のみが一覧に表示されることを検証する。
・任意の議事録を選択した際、右側の本体エリアで要約・決定事項・アクションアイテムが正しく表示されることを確認する。
・録音や編集用のコントロールがUIから除外されていることを確認する。
・データが存在しない場合や取得失敗時に適切なメッセージやローディング表示が出ることをテストする。
・アプリ内のナビゲーション等から「議事録一覧」ページへ遷移できることを確認する。

# Subtasks:
## 1. Supabaseから議事録・関連アイテムデータ取得APIの実装 [done]
### Dependencies: None
### Description: Supabaseのテーブルから過去の議事録や関連アイテム（セッションデータ）を取得するAPI呼び出しを実装する。認証なしでアクセス可能とし、データ取得中のローディング表示や空状態メッセージも考慮する。
### Details:
Supabaseクライアントを用いて、指定テーブルから必要なカラム（タイトル、日付、要約、決定事項、アクションアイテム等）を取得するAPIを実装する。データ取得時のエラーハンドリングやローディング・空状態UIも含める。

## 2. 2カラムレイアウトUIの構築とサイドバーリスト表示 [done]
### Dependencies: 14.1
### Description: フロントエンドで2カラムレイアウトを実装し、左サイドバーに日付とタイトルの一覧をリスト形式で表示する。
### Details:
取得したデータをもとに、左側に日付・タイトル一覧をリスト表示するUIを作成する。選択状態の管理やスクロール対応も考慮する。

## 3. 議事録詳細表示エリアの実装 [done]
### Dependencies: 14.2
### Description: 右側の本体エリアに、選択した議事録の要約・決定事項・アクションアイテムを見やすく整理して表示するUIを実装する。
### Details:
サイドバーで選択された議事録の詳細情報（要約・決定事項・アクションアイテム）を右側エリアに表示し、情報の区切りや見やすさを意識したレイアウトにする。

## 4. フロントエンドワード検索機能の実装 [done]
### Dependencies: 14.2
### Description: 左サイドバー上部にワード検索ボックスを設置し、タイトル・要約・決定事項・アクションアイテムのいずれかに該当する議事録のみを絞り込めるようにする。
### Details:
入力されたキーワードでリストをリアルタイムにフィルタリングし、該当する議事録のみを表示する。検索対象フィールドはタイトル・要約・決定事項・アクションアイテムとする。

## 5. アプリ内ナビゲーションへの「議事録一覧」導線追加とUIリファクタリング [done]
### Dependencies: 14.3, 14.4
### Description: アプリ内のナビゲーションやメニュー等に「議事録一覧」ページへの導線を追加し、必要に応じて既存UIコンポーネントのリファクタリングを行う。
### Details:
グローバルナビゲーションやサイドメニュー等に「議事録一覧」へのリンクを追加し、UIの一貫性や再利用性向上のために既存コンポーネントのリファクタリングも検討・実施する。
